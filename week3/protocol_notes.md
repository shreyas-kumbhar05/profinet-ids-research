# What is PROFINET

- PROFINET is an open Industrial Ethernet solution based on International standards
- It is a communication protocol designed to exchange data between controller and devices in an automation environment
- PROFINET products includes PLCs, PACs, Drives, robots, proxies, diagnostics tools, etc	


# Components of PROFINET

### IO Controller

- A controller is the brain of the system it makes decisions based on data
- It determines start and stop motor, open and close valve, emergency shutdown, etc

### IO Device

- Profinet device includes remote I/O module, motor drive, valve controller, sensor module, robot controller, etc
